Webapp bootstrap
==================

Cool maven skeleton used as a rapid starting point on Java/Kotlin web aplications

* Struts and Tiles for MVC
* Jersey for REST
* Shiro for Security

To test API Security:

    curl -H "X-ApiKey: test" http://localhost:8080/api/v1/hello
